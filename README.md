AirCanvas Pro is a gesture-controlled virtual drawing application built with Python, OpenCV, and MediaPipe. It allows users to draw in the air using finger movements captured by a webcam. The system supports real-time hand tracking, tool selection through gestures, multiple brush colors, erasing, saving, and smoothing for stable interaction.

*> Features :* 

• Hand Pen tracking using webcam

• Draw in air like a real canvas

• Multiple color selection

• Eraser mode

• Real-time performance

• Built with OpenCV
